# Neovim Configuration

This is my personal Neovim Configuration that I put together.
It's built on Lazy.nvim as the package manager, and much of what folke has put out.

## Semantics

| Key | Meaning |
|-----|---------|
| `o` | Open    |
| `t` | Toggle  |
| `f` | Find    |
| `c` | Code    |
| `w` | Window  |
